# Blinkit Sales Analysis Dashboard

## Project Overview

This project is an interactive Power BI dashboard developed to analyze Blinkit's sales performance, outlet performance, customer ratings, and product category trends. The dashboard provides actionable insights that can help improve sales strategies, inventory planning, and outlet performance.

## Objectives

* Analyze overall sales performance across outlets.
* Evaluate outlet-wise sales contribution.
* Monitor customer ratings and product performance.
* Identify high-performing item categories.
* Compare sales across outlet sizes and locations.

## Dashboard Highlights

### Key Performance Indicators (KPIs)

* Total Sales: $1M+
* Average Sales: $141
* Average Ratings: 4
* Number of Items: 9K+

### Visualizations Included

#### 1. Sales by Outlet Establishment Year

* Tracks outlet sales performance over time.
* Helps identify growth trends and peak-performing years.

#### 2. Sales by Item Type

* Displays sales contribution from different product categories.
* Highlights top-selling categories.

#### 3. Outlet Fat Content Analysis

* Compares sales generated by Low Fat and Regular products.
* Provides insights into customer purchasing preferences.

#### 4. Sales by Outlet Size

* Analyzes sales performance across Small, Medium, and High-sized outlets.

#### 5. Sales by Outlet Location Type

* Compares Tier 1, Tier 2, and Tier 3 outlet performance.
* Helps identify the most profitable market segments.

#### 6. Outlet Type Performance Table

* Displays:

  * Total Sales
  * Average Sales
  * Average Ratings
  * Number of Items
* Enables outlet-type comparison.

## Business Insights

* Tier 3 outlets generated the highest sales contribution.
* Medium-sized outlets outperformed other outlet sizes.
* Fruits, snacks, and household products contributed significantly to sales.
* Customer ratings remained consistently strong across outlet types.
* Low-fat products showed strong demand alongside regular products.
